Connexion à GitHub

Pour enregistrer tes modifications dans le dépôt, colle ici un token GitHub personnel ayant le droit d'écrire sur le dépôt visé ci-dessous. Il reste uniquement dans ton navigateur, il n'est jamais envoyé ailleurs qu'à l'API GitHub.

  1. github.com → Settings → Developer settings → Fine-grained tokens
  2. Repository access → sélectionne le(s) dépôt(s) concerné(s)
  3. Permissions → Contents : Read and write
  4. Génère puis colle le token ci-dessous
Laisse ce champ vide pour réutiliser le token déjà utilisé lors de ta dernière connexion (si tu changes juste de dépôt, mais que ton token a accès aux deux).
